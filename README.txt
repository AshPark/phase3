README

1. Open Xampp control panel
2. start Apache and MySQL
3. Upload phase3dat.sql to PhpMyAdmin.  It will create a database named hw3
4. Create a user in PhpMyAdmin with these specifications. 
   Username = root, Password = 1234, Full Permissions
5. In a web browser using localhost navigate to and run the file goats.php inside the folder phase3.  

In our experience it was easiest to store the folder phase3 in xampp's prefered location.
